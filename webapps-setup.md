Web Apps for Meta Ray-Ban Display glasses
Overview
Web Apps are an easy and familiar way to build and test unique experiences for Meta Ray-Ban Display (MRBD) glasses, especially with AI-assisted coding tools. Web Apps for Meta Ray-Ban Display glasses are standard HTML/CSS/JavaScript applications rendered on the glasses  built-in display.
This guide will help you build Web Apps optimized for the display, input, and device capabilities of Meta Ray-Ban Display glasses (MRBD).
Web Apps currently have access to:
MRBD display
Input signals from the Meta Neural Band and MRBD Cap Touch controls
Device motion/orientation data like accelerometer, gyroscope, and compass
User location, as provided by connected mobile devices
Local storage
