Single components show the pieces. These show the rhythm: how Navigator,
focus memory, the back gesture, and one-task-per-view compose into a real app.
Arrow keys move the ring, Enter pinches, Escape is the back gesture. Each QR
installs the example as a standalone app on real glasses.
