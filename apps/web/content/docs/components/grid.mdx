## When to use

Reach for **Grid** when every cell is the same shape and you want them aligned in
neat rows: a grid of square thumbnails, a launcher-like set of equal cards. Pass
`columns` (2, 3, or 4) and it lays them out on equal tracks, scrolling
vertically.

If the items vary in height and you would rather pack them tight without the
rows lining up, use [Masonry](/docs/components/masonry). Grid is layout only: the
children own their interactivity (a [MediaThumb](/docs/components/media-thumb)
with `onSelect`, or a [Pressable](/docs/components/pressable)).
